Commandline scripts

AaegL1/2/3>Juneja VCF Converter

Description: Converts vcf file aligned to AaegL1/2/3 (superconts) into mapped positions on chromosomes according to Juneja et al. 2014. Supercontigs that were not able to be mapped are left as is. Positions are then sorted by chromosome so chr1-3 come first, followed by unmapped chromosomes.

Usage: python variantpositions.vcf

Input: Standard VCF file where header rows commence with ## and the first two columns are CHROM and POS based on positions aligned to AaegL1-3. Also required is the supplemental file outlining mappings between supercontigs and chromosomes called JunejaGeneticAssembly.csv

Output: Copy of the given VCF file with prefix juneja.

Download: AaegyptVCFConverter.py & JunejaGeneticAssembly.csv

UniProt&BioMart>Gowinda

Description: Downloads latest Ae. aegypti associated Gene Ontology terms from UniProt & BioMart using their REST APIs and creates input to the program Gowinda (publication)

Usage: python UniprotToGowinda.py

Depends: Requires the python module wget to be installed. Should be as simple as using the command: pip install wget in the terminal if you don't have it.

Input: Optional to supply already downloaded files from Uniprot and GO consortium using the flags:

--gene_file [uniprotoutput.tab.gz]

--obo_file [go.obo]

Output: Tab-delimited text file entitled UniprotBioMartGOassocGowinda.txt ready for input into Gowinda, eg:

GO:0007005 mitochondrion_organization AAEL007089 AAEL010831 AAEL001853 AAEL003829 AAEL011606
GO:0006228 UTP_biosynthetic_process AAEL003030 AAEL004107 AAEL011098 AAEL012359

Download: UniprotToGowinda.py
