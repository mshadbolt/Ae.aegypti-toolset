This folder contains Python code for simulating populations using the package simuPOP, for Pritchard et al. (2016), plus additional data files.

*_beta.py, Naatamo.py, Tornio.py : code for simulating test individuals.
*_ref_beta.py, Naatamo_ref.py, Tornio_ref.py : code for simulating reference individuals.

Estimated_allele_frequencies.csv: estimated frequencies of the 200 discriminatory alleles within populations, from Table S3.
Allele_frequencies_for_simulations.csv: adjusted allele frequencies used for simulations, from Table S3.
